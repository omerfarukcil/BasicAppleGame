A fast-paced 3D game built with Unity and C#. Players must collect a mysterious apple to unlock the exit, but there's a catch: the moment you pick it up, the enemies start hunting you down!

Gameplay Mechanics
The Trigger: The game starts peacefully, but picking up the Golden Apple sets the isAppleCollected flag to true.

The Chase: Once the apple is collected, the LevelManager alerts all enemies to start chasing the player.

The Goal: Reach the Exit Door before the enemies catch you to complete the level.
